# Nagpur For Good App

Frontend application for the Nagpur For Good platform.

## Features

- Home page with hero, search, and category filtering
- NGO cards and animated grid layout
- NGO detail pages with impact timeline and transparency links
- Donation modal and polished motion interactions
- SEO metadata via React Helmet Async

## Stack

- React 19
- Vite 7
- Tailwind CSS 4
- Framer Motion
- React Router DOM 7
- Lucide React

## Scripts

```bash
npm run dev      # start local dev server
npm run build    # production build
npm run preview  # preview production build
npm run lint     # run ESLint
```

## Setup

```bash
npm install
npm run dev
```

## Note on Node version

Vite 7 requires Node.js `20.19+` (or `22.12+`).
